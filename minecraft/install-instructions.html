Installation Instructions for Technic Modpacks

1. Download and install the version of Java that your system uses: Windows - macOS - Linux

2. Download the version of the Technic Launcher that runs on your system (downloads at bottom of page): Technic Launcher

3. Run the Technic Launcher and click Install, don't modify any settings when installing.

4. Login using your Minecraft login credentials.

5. Click the Modpacks tab, and then click Launcher Options, then Java Arguments.

6. Look at the RAM listings available. If your computer's maximum setting is 3.5GB, go with that. If your maximum setting is 7GB, go 4GB. If your maximum setting is 16GB, go 12GB. After setting the value, exit out of the options box.

7. Type in the name of the modpack you wish to use. After selecting a pack, click Install, and then click Play.

Tekkit Classic Instructions To access the Tekkit Sprite server, you will need to install Tekkit 3.1.3. Here's how:

1. Install Tekkit Classic as you usually would, using the above method. Do not click play when it is installed.

2. Click Modpack Options and change the version by clicking specific version and selecting 3.1.3. Click "Reinstall Pack", "Yes" on the dialogue box, and then close the modpack options menu. and then reinstall Tekkit Classic. Do not click Play.

3. After Tekkit Classic is reinstalled, click Modpack Options and then click Open. Navigate to the mods folder in the window that opens.

4. Delete the ComputerCraft folder. There is also a file called ComputerCraft1.41, leave it alone. Now, go up a level back to the main tekkit folder.

5. Go into the bin folder, download this and replace the modpack.jar file in the folder. This will enable skins to load on 1.2.5.

6. You can now close all open non-Technic windows, close the modpack options menu, and click Play. Ignore all requests to update Tekkit Classic, if you click Yes, the launcher will reinstall 3.1.2.
