Installation Instructions for Technic Modpacks

1. Download and install the version of Java that your system uses: Windows - macOS - Linux

2. Download the version of the Technic Launcher that runs on your system: Windows - macOS - Linux

3. Run the Technic Launcher and click Install, don't modify any settingss when installing.

4. Login using your Minecraft login credentials.

5. Click the Modpacks tab, and then click Launcher Options, and then Java Arguments.

6. Look at the RAM options available. If you don't see anything about 1.5GB, you are running the 32-bit version of Java and need to download the 64-bit version from the above links. If you have a maximum option of 3.5GB, select 2GB. If you have a maximum option of 7GB, select 4GB. If your maximum available option is 15GB, select 8GB. If you want to use a big resource pack such as PureBDcraft, increase this to 10 or 12. After setting this value, exit out of the options box.

7. Type in the name of the modpack you wish to use. After selecting a pack, click Install, and then click Play.
