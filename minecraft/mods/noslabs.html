NosLabs Custom Blocks

NosLabs Custom Blocks was born after Sean grew tired of changing
resource packs when going between NosLabs and the surface world.

The source code is available on GitHub.

Release 3.0.0, issued on April 21, 2019, is a massive rewrite of the mod from top to bottom. It should not corrupt any worlds, as we made sure the block registry names were the same, and it adds tons of new things. You will need to update the HD texture pack if you use it.

Since March 22, 2019, the mod now contains the original low-resolution textures of TheVoxelBox Scientific.
If you want the HD textures, download this: NosLabs HD Resource Pack. You will also need OptiFine to enable connected glass textures.
