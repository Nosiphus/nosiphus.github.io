OptiFinder

OptiFinder is a clientside mod that reminds users to manually install OptiFine before loading their modpacks.
If it does not detect OptiFine, it will not allow the modpack to continue loading.
This mod was created to satisfy sp614x's rules on modpack inclusion.

The source code is available on GitHub.
