Installing OptiFine 1.16.2_HD_U_G3_pre1 for 1.16.2 Modpacks

There is a dialog box that should be either behind or on top of the main Minecraft window. You will need this box to proceed.

1. Click the "Open Directory" button on the dialog box.

2. Go to the official download page to download the OptiFine installer. DO NOT place this in the "Open Directory" folder.

3. If you have not already done this in the past, load the vanilla Minecraft launcher and install an instance of Minecraft 1.16.2. Once you have loaded the game once, you can close it and skip this step in the future.

4. Double-click the OptiFine installer, and then click Extract. Extract the file to the desktop, and do not rename the file.

5. Move the OptiFine_1.16.2_HD_U_G3_pre1_MOD.jar file into the "Open Directory" folder.

6. Click Play to reload the modpack with OptiFine now installed. If the pack fails to launch, restart your computer.
