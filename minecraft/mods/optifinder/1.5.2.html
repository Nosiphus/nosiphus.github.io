Installing OptiFine for 1.5.2 Modpacks

OptiFinder is currently not available for Minecraft 1.5.2, but it will be soon.

Installing OptiFine on Minecraft versions before 1.6 is a bit trickier. If this instruction set is above your paygrade, email Nosiphus Support.

If you see this window and not the dialog box, the box should be behind the main Minecraft window, which should be behind this browser window.

1. Click the Open Directory button on the dialog box that appears.

2. Go to this website to download OptiFine.

3. Install WinRAR.

4. Open the OptiFine zip with WinRAR. Also, right-click "modpack.jar" and click Open with WinRAR.

5. Drag all files inside the OptiFine zip file into modpack.jar.

6. Close all of these windows and relaunch the modpack.
