Installing OptiFine for 1.6.4 Modpacks

If you see this window and not the dialog box, the box should be behind the main Minecraft window, which should be behind this browser window.

1. Click the Open Directory button on the dialog box that appears.

2. Go to this website to download OptiFine. Place this in the "Open Directory" folder.

3. Click Play to reload the modpack with OptiFine installed. If the pack fails to launch after this, restart your computer.
