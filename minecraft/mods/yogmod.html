YogMod

YogMod, formerly the NosLabs Custom Blocks mod, was born after Sean grew tired of changing
resource packs when going between NosLabs and the surface world.

The source code is available on GitHub.

YogMod is NOT compatible with NosLabs Custom Blocks mod-based worlds. If you want to keep the same world, you will need to use WorldEdit's replace function with both mods installed.

Since March 22, 2019, the mod now contains the original low-resolution textures of TheVoxelBox Scientific.
High Definition Textures
