The Nosiphus Required Mods are designed for Minecraft 1.5.2. They are available on the Technic Launcher.
The current release of the Nosiphus Required Mods modpack is version 1.2.0, released on Wednesday, July 17, 2013.

Custom Sphax PureBDcraft Patches
Sphax PureBDcraft Modded Textures for Nosiphus Required Mods - May 24, 2016 for NRM 1.2.0+
Sphax PureBDcraft for Minecraft 1.5.x
Patch the main Sphax PureBDcraft by placing the modded folders alongside the main patch folders. Allow the mods to overwrite existing textures.

Basic Prerequisites
Technic Launcher
Java Runtime Environment 7u80 x64 - Windows - macOS - Linux
JRE 7 is required if you want to run an MCPC+ server on Minecraft 1.5.2.

This pack consists of the latest 1.5-compatible versions of the original collection of mods used during the summer of 2013. It carries the date of the original file's creation.
